基于 `Flurl.Http` 的又拍云云存储服务（USS）HTTP API SDK。

---

### 【功能特性】

-   基于又拍云云存储服务 API 封装。
-   支持 REST API、FORM API 两种上传模式。
-   请求时自动生成认证签名，无需开发者手动干预。
-   提供了又拍云云存储所需的 MD5、HMAC-SHA-1 等算法工具类。
